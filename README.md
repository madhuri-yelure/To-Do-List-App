# To-Do List App

A responsive To-Do List web application built using HTML, CSS, and JavaScript for efficient task management. It allows users to add, complete, and delete tasks through a clean and user-friendly interface.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Displays total, completed, and remaining tasks
- Responsive and user-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript
